# SunstormAI — Project Context

## Brand
- Name: SunstormAI — AI automation agency → SaaS platform
- Style: Crimson Core + Void Protocol hybrid. Full intensity. No softening.
- Personality: Volcanic. Precise. Unapologetic. Premium dark tech.

## Colours
- --storm-red: #C8142A (primary accent, all CTAs)
- --storm-red-hot: #FF4060 (hover, active states)
- --storm-black: #0D0608 (primary dark background)
- --storm-black-mid: #1A0810 (surface / cards)
- --storm-void: #08080A (deepest background)
- --storm-white: #FAF8F6 (light surface)

## Fonts
- Space Grotesk — headings and display
- Inter — body copy
- JetBrains Mono — labels, code, mono accents

## Rules
- "storm" in the wordmark is always #C8142A
- Never soften the dark theme
- No frameworks, no build step — vanilla HTML/CSS/JS only
- All files saved to project root unless told otherwise
- Google Fonts imported in every HTML file
- Fully responsive, mobile first
